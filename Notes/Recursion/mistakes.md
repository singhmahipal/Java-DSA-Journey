
# ❌ Mistakes & Struggles - Recursion

---

## 🚫 Mistake 1: Forgot Base Case

**Problem:** Infinite recursion / stack overflow  

**Why it happened:**  
I jumped into recursion without defining stopping condition  

**Fix:**  
Always write base case FIRST  

---

## 🚫 Mistake 2: Wrong Recursive Call

**Problem:** Wrong output  

**Example:**  
Used `f(n)` instead of `f(n-1)`

**Why it happened:**  
Did not reduce problem size  

**Fix:**  
Ensure input becomes smaller every call  

---

## 🚫 Mistake 3: Didn’t Understand Flow

**Problem:** Couldn’t trace output  

**Why it happened:**  
Did not visualize recursion tree  

**Fix:**  
Draw recursion tree for every problem  

---

## 🚫 Mistake 4: Fear / Confusion

**Problem:** Couldn’t start problem  

**Why it happened:**  
Trying to solve everything at once  

**Fix:**  
Think:
👉 What will function do for n?  
👉 Assume it works for n-1
